A fast-paced, neon-themed version of the classic Snake game. This project is built using a single-file architecture (HTML/CSS/JS) to demonstrate efficient game-loop logic and canvas rendering.

 Features
* **Auto-Start Logic:** The game begins immediately upon loading for a seamless experience.
* **Neon Glow UI:** Custom CSS and Canvas Shadow API used to create a high-contrast "Cyberpunk" aesthetic.
* **Collision Engine:** Handles wall collisions and self-intersection to manage game-over states.
* **Score Tracking:** Real-time score updates as the snake consumes food.

 Technologies Used
* **HTML5 Canvas:** For high-performance 2D rendering.
* **CSS3:** For the dark-mode layout and neon glowing borders.
* **JavaScript (ES6):** * `setInterval` for the 100ms game tick.
    * Keyboard Event Listeners for directional control.
    * Array manipulation for snake growth and movement.

 How to Play
1. **Open the Project:** Launch `index.html` in any modern web browser.
2. **Controls:** Use the **Arrow Keys** (Up, Down, Left, Right) to navigate.
3. **Objective:** Eat the pink food blocks to grow your snake and increase your score without hitting the walls or your own tail.
